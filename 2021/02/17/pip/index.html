pip 使用入门

2021-02-17 python 535 字
python pip

介绍

pip是python自带的包管理工具，它实现了对Python第三方包的管理功能。

安装

一般来说，如果你在python中下载了python的最新版本（python3.4+），则已经自带了该工具。

检测

在 cmd 中输入

pip -V

  • 如果返回了

    pip 20.2.4 from ···s

则说明安装成功；

  • 如果返回了

    'pip' 不是内部或外部命令，也不是可运行的程序或批处理文件。

则说明在安装python的时候没有选中 Add python3.x to path 属性，建议重新进行python的安装，或者可以通过命令行单独安装pip。

1. python的重新安装

这是比较推荐的方法，还可以更新电脑上的python

  1. 首先，点击这里下载python3.9的安装包。
  2. 在第一个界面，点击界面下方Add Python 3.9 to PATH

  1. 然后点击Install Now


此处不建议修改默认安装路径
4. 之后就可以愉快的无脑下一步啦~
5. 最后，打开cmd检测pip是否安装完成就可以了

2. 通过命令行单独安装pip

此过程比较简单方便，对命令行命令有兴趣的同学可以自行搜索了解相关命令的作用

  1. Win + R打开运行窗口，输入cmd打开终端

  2. 在cmd中输入

    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

  3. 下载完成后，继续输入

    python get-pip.py

  4. 完成并检测后就可以愉快的使用pip啦~

使用

目前，我们最需要用到的pip命令只有一个：

pip install SomePackage

即用pip自动安装python第三方包。

举个栗子：

pip install pygame

就可以安装python一个简单的小游戏制作工具包 pygame

可能用到的命令

  1. pip install SomePackage                 //安装指定包
  2. pip uninstall SomePackage               //卸载指定包
  3. pip --version                           //查看pip版本
  4. pip --help                              //获取pip帮助
  5. python -m pip install --upgrade pip     //升级pip
  6. pip list                                //列出已安装的包
  7. pip list -o                             //列出可升级的包
  8. pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
//将pip镜像源改为清华，能够提升包下载速度
